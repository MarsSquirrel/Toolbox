# Installation

### Method 1 - Manual

1. Visit the [latest release](https://github.com/MarsSquirrel//releases/latest)
2. Under *Assets*, download `Toolbox.rbxm` or `Toolbox.rbxmx`
3. Insert the file into your game.

### Method 2 - Package Manager

1. Setup [Wally](https://wally.run/) by running `wally init`.
2. Add `marssquirrel/toolbox` to your dependencies

```toml
Toolbox = "MarsSquirrel/toolbox@^1.0.0"
```
